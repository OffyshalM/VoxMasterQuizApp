# VoxMasterQuizApp
VoxMaster is a dynamic, voice-enabled quiz platform designed to enhance accessibility in digital learning. Unlike traditional quiz apps, VoxQuery integrates the Web Speech API to narrate questions and options, providing an inclusive experience for users with visual impairments or those who prefer auditory learning.
Key Technical Features:
Speech Synthesis Integration: Leverages the native browser window.speechSynthesis interface to convert text into natural-sounding speech without external API latency.

Dynamic Content Management: Uses PHP to securely fetch and serve quiz data from a MySQL database, allowing for easy updates and category management.

Asynchronous Interaction: Implemented JavaScript (AJAX/Fetch) to handle quiz logic and score calculations without page reloads, ensuring a smooth, app-like feel.

Mobile-First Design: Crafted with responsive CSS to ensure the "Listen" and "Answer" controls are perfectly scaled for one-handed thumb use on mobile devices.
